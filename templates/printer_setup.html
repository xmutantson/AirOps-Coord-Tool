Label Printer Setup

The Air Ops tool prints inventory tags and cargo labels to a Brother QL-820NWB label printer using 62mm (DK-2205) continuous tape. Follow these steps to set up your computer.

Step 1: Install the printer driver

Windows does not include the Brother QL driver by default. Download and run the installer, then follow the prompts.

Download Brother QL-820NWB Driver (Windows)

34 MB · Also available from Brother support site

  1. Run the downloaded Brother_QL820NWB_Driver.exe
  2. Accept the license agreement and click Install
  3. When prompted for connection type, choose Network (Wi-Fi)
  4. The installer should auto-detect the printer on the network
  5. If it does not, enter the printer IP manually (ask your Ramp Boss)
  6. Finish the installer and print a test page to confirm

Step 2: Configure printer defaults

After installing, you need to set the correct paper size so labels print at the right dimensions. This only needs to be done once.

  1. Open Settings → Bluetooth & devices → Printers & scanners
  2. Click Brother QL-820NWB
  3. Click Printing preferences
  4. On the Basic tab, set these values:
    Setting Value Why
    Paper Size 2.4" x 3.9" 2.4" is the 62mm tape width. 3.9" is the longest preset — the auto-cutter trims to actual content length
    Width 2.4" Auto-filled from paper size
    Length 3.9" (or leave as-is) Maximum print area — the cutter handles the rest
    Feed 0.12" Small gap before cut line. Default is fine
    Orientation Landscape Labels are wider than tall
    Quality Give priority to print speed 300x300 DPI — fast and clean barcodes
    Cut Every Checked, 1 labels Cuts between each label
    Cut at end Checked Final cut after last label
    Mirror Printing Unchecked Leave off
    Trim tape after data Unchecked Not needed — "Cut Every" handles it
  5. Click Apply, then OK

Step 3: Direct Printing (Recommended)

Best option: When direct printing is enabled, the Air Ops server sends labels straight to the printer over the network. No driver install needed on your computer, no print dialog, one-click printing. The printer auto-cuts each label at exactly the right length.
  1. Go to Preferences (gear icon in the nav bar)
  2. Scroll to Label Printer
  3. Set Direct printing to On
  4. Enter the printer's IP address (ask your Ramp Boss)
  5. The status badge should show reachable
  6. Now clicking any label print button sends directly to the printer

With direct printing enabled, you do not need to install the Brother driver or configure paper sizes. The server handles everything.

Alternative: Browser Printing

If direct printing is not available (turned off, printer unreachable), labels will fall back to the browser print dialog. For this you need the driver installed (Steps 1-2 above).

  1. In the print dialog, select Brother QL-820NWB from the printer list
  2. Set Scale to 100% / Actual size (not "Fit to printable area")
  3. Set Margins to None if available
  4. Click Print

Troubleshooting

Printer not found during install
Make sure you are on the same Wi-Fi network as the printer. The QL-820NWB is configured in wireless infrastructure mode and should appear on the same subnet.
Labels print too small or too large
Check that the print dialog scale is set to 100% (not "Fit to page"). Also verify the Paper Size in printer preferences is set to 2.4" x 3.9".
Labels cut in the wrong place
Ensure "Cut Every 1 labels" and "Cut at end" are both checked in printer preferences. The Feed value should be around 0.12".
Barcode won't scan after printing
Set Quality to "Give priority to print speed" (300x300 DPI). Higher quality modes can cause barcode lines to bleed together on thermal tape.
