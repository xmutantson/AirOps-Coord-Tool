Label Printer Setup

The Air Ops tool prints inventory tags and cargo labels to a Brother QL-820NWB label printer using 62mm (DK-2205) continuous tape. Follow these steps to set up your computer.

Step 1: Install the printer driver

Windows does not include the Brother QL driver by default. Download and run the installer, then follow the prompts.

Download Brother QL-820NWB Driver (Windows)

34 MB · Also available from Brother support site

  1. Run the downloaded Brother_QL820NWB_Driver.exe
  2. Accept the license agreement and click Install
  3. When prompted for connection type, choose Network (Wi-Fi)
  4. The installer should auto-detect the printer on the network
  5. If it does not, enter the printer IP manually (ask your Ramp Boss)
  6. Finish the installer and print a test page to confirm

Step 2: Configure printer defaults

After installing, you need to set the correct paper size so labels print at the right dimensions. This only needs to be done once.

  1. Open Settings → Bluetooth & devices → Printers & scanners
  2. Click Brother QL-820NWB
  3. Click Printing preferences
  4. On the Basic tab, set these values:
    Setting Value
    Paper Size 62mm (pick any length — the app controls cut length)
    Width 2.4" (62mm)
    Orientation Landscape
    Quality Give priority to print speed
    Cut Every Checked, 1 labels
    Cut at end Checked
    Trim tape after data Unchecked
  5. Click Apply, then OK

Step 3: Printing from Air Ops

When you click a label print button in Air Ops, a print dialog will open. Make sure you select Brother QL-820NWB as the printer.

  1. In the print dialog, select Brother QL-820NWB from the printer list
  2. The label content is already sized correctly by the app — do not change scaling or margins in the print dialog
  3. Set Scale or Fit to page to 100% / Actual size (not "Fit to printable area")
  4. Set Margins to None if the option is available
  5. Click Print
Tip: Most browsers remember the last printer you selected. After choosing the Brother QL once, it should stay selected for future prints.

Troubleshooting

Printer not found during install
Make sure you are on the same Wi-Fi network as the printer. The QL-820NWB is configured in wireless infrastructure mode and should appear on the same subnet.
Labels print too small or too large
Check that the print dialog scale is set to 100% (not "Fit to page"). Also verify the Paper Size in printer preferences is set to 62mm.
Labels cut in the wrong place
Ensure "Cut Every 1 labels" and "Cut at end" are both checked in printer preferences. The Feed value should be around 0.12".
Barcode won't scan after printing
Set Quality to "Give priority to print speed" (300x300 DPI). Higher quality modes can cause barcode lines to bleed together on thermal tape.
